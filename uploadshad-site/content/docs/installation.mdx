### Overview

UploadShad is a powerful and easy-to-use file uploader component designed for ShadCN forms and React applications. It provides seamless integration, customizable options, and robust handling of file uploads.

##### Breakdown

- Install Shadcn and dependant components
- Install UploadShad dependencies
- Install UploadShad

<br />

<Steps>

### Run the CLI

Run the shadcn-ui init command and setup your project with Shadcn:

<br />

```bash
pnpx shadcn-ui@latest init
```

Install the **Form** and dependant Shadcn components by running the following command:

<br />

```bash
pnpm dlx shadcn-ui@latest add form progress aspect-ratio card
```

### Installation

Install the dependant modules

<br />

```bash
pnpm add @hello-pangea/dnd sonner react-dropzone sharp uuid @aws-sdk/client-s3
```

<small>AWS SDK is optional, remove HonoS3 if you don't wish to use it</small>

### Environment Variable

For UploadShad to be more user-friendly, it does expect a **AWS_S3_BASE_URL** environment to build urls for image previewing.
You can provide a non S3 base url, that matches your storage solution choice.

<br />

```
AWS_S3_BASE_URL=<your_storage_solutions_base_url>
```

### Install UploadShad

Install **UploadShad** by running the following command:
(Coming soon)

<br />

```bash
pnpm install uploadshad
```

<small>Or, use any package manager of your choice.</small>
