### Overview

The **DataTable** component is a powerful, flexible, and lightweight solution for rendering tabular data in your applications.
Built on top of [ShadCN's DataTable component](https://ui.shadcn.com/docs/components/data-table) which uses
[TanStack Table](https://tanstack.com/table), this component simplifies the process of creating dynamic tables with rich features.

##### Breakdown

- Install Shadcn and dependant components
- Install DataTable dependencies
- Install DataTable

<br />

<Steps>

### Run the ShadCN CLI

Run the shadcn-ui init command and setup your project with Shadcn.

<br />

```bash
pnpm dlx shadcn@latest init
```

### Add DataTable

This will add DataTable & install it's required dependencies into your project.

<br />

```bash
pnpm dlx shadcn@latest add https://uploadshad.nerfdesigns.com/registry/data-table.json
```

<small>This uses the (new) ShadCN CLI.</small>
